# Intentional Finance

### Category-Based-Budget-System-in-Excel

**Intentional Finance** is a personal finance tracking system built in Excel, inspired by the **cash stuffing** methodology. It is designed to help users manage their income and expenses through intentional allocation across categories, on a monthly basis.

---

## 🌿 Project Overview

This project functions as an **Expense Segmentation Model**, where financial data is structured and analyzed by category and time (month).

The goal is to provide a simple yet powerful system that encourages conscious spending and financial clarity where after a few months, the user has always money for every decided intention.

Unlike many modern finance tools, this system is intentionally built in Excel to ensure **accessibility**:
anyone with a computer or tablet and basic office software can use it — no need for specialized tools like Power BI.

---

## ❓ Problem It Solves

Most personal finance tools are either:

* Too complex or overwhelming
* Locked behind specific platforms or software
* Not designed for intentional, category-based budgeting

This project solves that by providing:

* A **clear structure for tracking income and expenses by category**
* A **monthly overview of financial behavior**
* A system that is both **accessible and easy to maintain**

---

## 💸 Methodology: Cash Stuffing

The system is based on the **cash stuffing (envelope budgeting)** method, where income is distributed into predefined categories (e.g., food, rent, savings, etc.) and includes some more extrvagant, real-life unexpected expenses as gifs, emergencies or simply, desire self-love gestures as buying yourself a pair of shoes.

Each category acts as a “digital envelope,” allowing users to:

* Allocate funds intentionally
* Track expenses per category
* Monitor monthly financial behavior

---

## 🧠 Data Structure

The core of the model is a structured table that includes:

* **Date (Month)**
* **Category**
* **Monthly Income**
* **Expenses per Category**

Using Excel functions such as `SUMAR.SI.CONJUNTO`, the model aggregates expenses dynamically and calculates remaining balances per month.

---

## 📊 Decisions Enabled by the Model

This system supports key financial decisions such as:

* How much money to allocate to each category at the start of the month
*  Identifying which categories consume the highest percentage of income
*   How much more do you need to have as income to increment the percentage for each category
* Evaluating whether current spending aligns with financial priorities

---

## 🔄 Decisions That Evolve with Monthly Insights

Based on the monthly dashboard, users can adjust their behavior over time:

* Reallocating budget between categories (e.g., reducing dining, increasing savings)
* Identifying overspending patterns and correcting them
* Adjusting financial goals based on real data
* Improving future monthly planning using past performance
